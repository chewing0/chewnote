## 整理约定

- `docs/` 优先保留“长期有效”的信息，不再堆积大量一次性专题记录。
- 某个专题如果只是一次短期优化，优先并入 [project-overview.md](/D:/wby/MyApplication/docs/project-overview.md) 的“最近迭代”或“页面现状”部分。
- 只有当某个模块后续会长期独立演进时，才单独拆文档。
